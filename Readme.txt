Project Title - MIND MORPHERS 

	It is a quiz game based on LCU(Lokesh Cinematic Universe) Movies.
	Your marks will transform into a character which is from LCU Movies.

About the project:
	Frontend: HTML, CSS, JavaScript.
	Backend:  JavaScript.

To this project:
	
	1.Open home.html file.
	2.Click let's start button.
	3.Then, you have enter your name.
	4.Now,quiz will be started.
	5.You have to attend 10 questions which is asked based on LCU movies.
	6.At end of the quiz, your name, score & a character.
